# AI Resume Builder

🚀 A web app that generates professional resumes using AI.

## 🔥 Features
- Generate resume from user input
- Professional formatting
- Clean and modern UI
- AI-enhanced content

## 🛠 Tech Used
- Google AI Studio
- Cloud Run
- JavaScript / Web UI
